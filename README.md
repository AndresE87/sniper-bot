## Sniper Bot

- dxsale
- pancake
- uniswap-v2

## usage

- download from [releases page](https://github.com/fitzix/sniper-bot/releases)
- run `sniper-bot init` generate config file
- fill the config.yml
- run cmd
    - dxsale `sniper-bot dxsale`
    - pancake `sniper-bot cake`
    - uniswap-v2 `sniper-bot uni`
